Hierarchy

The brief for this project was deliberately arbitrary. For our Interaction Design final, each group was handed three randomly assigned words: a shape, a color, an emotion and told to build something from them. We got triangle, bright, and tired.

We prototyped several directions and landed on a particle system. Each particle, representing people perpetually jumping upward toward the top.

The user has some control, and it is up to them to either make the system easier or difficult to progress. You can add more balls to the system, push them around, raise or lower the goal to make the it impossible or easy to achieve.

We didn't want to make this project into a game as games imply the possibility of winning. This is a visual experience about what it feels like to be inside a system that only requires you to keep going, in a bright, organized and colorful world.
